# lazycat-mcp
懒猫微服mcp - 使用本服务可以实现懒猫微服硬件和大模型的交互。

## Roadmap
### 基础能力
- [ ] 文件读写
### 硬件
- [x] 电源灯状态查询、开启和关闭
- [x] 设备关机和重启
- [x] 设备列表获取
- [x] 设备通知推送
### 数据及应用
- [ ] 应用管理
- [ ] 应用商店数据获取及建议
### 交互
- [x] 懒猫设备通知推送
- [ ] bark数据推送
- [ ] 邮件数据推送
- [ ] 其他数据推送
## 使用说明
 Cherry Studio中添加mcp服务器即可,其他客户端请参考对应的使用说明文档.
